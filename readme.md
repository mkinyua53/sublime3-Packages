## About this repo
This is my sublime text 3 packages. 
The folder is located in '~UserFolder~\AppData\Roaming\Sublime Text 3'

### Included Packages
- All AutoComplete
- Laravel Blade Highlighter
- Emmet
- SideBar Enhancement
- VueJs Syntax Highlighter

### Settings
- tab_size: 4,
- translate_tab_to_space: true,
- save_on_focus_lost: true
